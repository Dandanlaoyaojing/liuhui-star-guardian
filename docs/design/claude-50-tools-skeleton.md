# Claude 版 50 关思维工具骨架

> 纯骨架。不含场景、不含诗意、不含交互细节。
> 每一行回答一个问题：**这一关，玩家练的是什么工具？玩完能说出什么话？**

---

## 第一阶段：基本工具（M01-M10）

> 单一认知操作。每关只练一把刀。

| 编号 | 思维工具 | 一句话定义 | 玩完能说的话 |
|------|---------|-----------|-------------|
| M01 | 分类与归纳 | 按属性把事物分成组 | "我会先看东西有什么共同点，然后分堆" |
| M02 | 模式识别 | 在混乱中找到重复的规律 | "乱的东西里其实藏着重复的规律" |
| M03 | 参照校准 | 先找到可靠的基准，再判断其他事物 | "判断之前，先确认自己的尺子准不准" |
| M04 | 路径规划 | 在约束条件下设计从A到B的最优路线 | "不能直接走的时候，绕路也是一种策略" |
| M05 | 间接策略 | 不直接作用于目标，通过改变环境来达成 | "有时候推水比推石头更聪明" |
| M06 | 排除法 | 通过去掉不可能的选项来缩小答案范围 | "找不到对的，就先去掉错的" |
| M07 | 序列排序 | 按规则把事物排成正确的顺序 | "顺序不对，结果就全不对" |
| M08 | 多条件满足 | 同时满足多个不同类型的约束 | "不是只满足一个条件，而是所有条件都要同时满足" |
| M09 | 流程优化 | 找到执行多个步骤的最佳顺序 | "做事的顺序比做事本身更重要" |
| M10 | 开放探索 | 没有唯一正确答案，自主定义"好"的标准 | "有时候没有标准答案，我自己觉得好就是好" |

> 把目光从外部世界收回，审视和调整自己的思维过程。

| 编号 | 思维工具 | 一句话定义 | 玩完能说的话 |

| M33 | 模因识别 | 传播最广的想法不一定是最正确的，而是最容易传播的 | "流行不等于正确" |

## 五阶段递增逻辑总览

```
第一阶段：一把刀    → 学会用单个工具
第二阶段：工具箱    → 学会组合多个工具
第三阶段：仪表盘    → 学会在动态系统中做决策
第四阶段：镜子      → 学会观察和调整自己的思维
第五阶段：锻造炉    → 学会重新铸造工具本身
```

## 自检标准

每一关必须通过这三个测试：

1. **工具可命名**：能用一个短语说出这关练的工具（如"排除法""反馈循环"）
2. **工具可复述**：玩完后玩家能用一句大白话说出学到了什么
3. **工具可迁移**：这个工具在日常生活中有明确的应用场景
