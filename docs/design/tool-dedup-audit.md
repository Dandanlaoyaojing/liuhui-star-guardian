## 二、重复族识别

### 族 A：分类/匹配/分拣 — "观察属性 → 归组 → 放对地方"

这是最大的重复族。**10 关做的是同一件事**。

| 关卡 | 工具名 | 和 M01 的真实区别 | 判定 |
|------|--------|-----------------|------|
| **M01** | 分类与归纳 | — | ✅ **保留（族代表）** |
| M02 | 模式匹配与排除 | 分两组而非多组 | ❌ 删（二元分类 = 分类特例） |
| M04 | 路径规划与分拣 | 加了空间载体 | ❌ 删（"路径"只是分类的投递方式） |
| M06 | 模式识别与节奏同步 | 属性换成时间节奏 | ❌ 删（节奏 = 属性的一种） |
| M09 | 流程优化与顺序逻辑 | 排因果顺序 | ⚠️ 见族 G |
| M12 | 属性分离与提纯 | 逆向分类 | ❌ 删（混合→分开 = 反向分类） |
| M16 | 优先级排序与取舍 | 排优先级 | ❌ 删（= 赋权后排序，与 M13 重叠） |
| M17 | 对称与互补思维 | 找另一半 | ❌ 删（配对 = 二元匹配） |
| M20 | 分工与协作优化 | 按能力分配 | ❌ 删（按特长匹配 = 分类后分配） |
| M32 | 符号、语义与翻译 | 跨符号映射 | ⚠️ 边界（"翻译"有独立性，但核心操作仍是映射匹配） |

**删 7 关**：M02, M04, M06, M12, M16, M17, M20
**边界 1 关**：M32（建议保留，因为"意识到同一个意思可以有不同表达"是独立洞察）

## 四、13 个空位该填什么？

以下是当前 50 关**完全缺失**的、真正独立的认知操作：

| # | 建议新工具 | 底层操作 | 为什么独立（不是现有工具的变体） |
|---|-----------|---------|------|
| N01 | **分解与重组** | 把大问题拆成可独立解决的小问题，再组装回去 | 现有工具都处理"整体"，缺乏"拆解"操作 |
| N02 | **逆向推理（从终点倒推）** | 从目标状态反向追踪到起始状态 | M11 是正向补链，这里是反向规划 |
| N03 | **抽象与具象化** | 剥离细节看本质 / 把抽象概念还原为具体实例 | M29 是空间维度，这里是信息密度维度 |
| N04 | **约束识别（找到真正的瓶颈）** | 在复杂系统中找到真正限制全局的那个约束 | M07 是"满足约束"，这里是"发现约束在哪" |
| N05 | **空间推理与心理旋转** | 在脑中旋转、折叠、变换空间结构 | 50 关完全缺乏纯空间认知操作 |
| N06 | **估算与近似** | 精确不可得时，做"足够好"的快速判断 | M26 是概率决策，这里是"精度换速度"的务实策略 |
| N07 | **否定与反证** | 通过证明"不是什么"来逼近"是什么" | M19 是正向验证，这里是反向排除 |
| N08 | **信息不对称识别** | 意识到"我不知道别人知道什么，别人不知道我知道什么" | 50 关几乎没有社会认知/心智理论维度 |
| N09 | **模型构建（简化现实）** | 为复杂现实搭建简化的心理模型来辅助理解和预测 | M40 是连接已有概念，这里是"从零造模型" |
| N10 | **边界测试（极端情况探测）** | 故意推到极端，看规则在哪里失效 | M38 是接受模糊，这里是主动探测边界 |
| N11 | **注意力分配与多任务管理** | 在多个同时进行的事务间有策略地分配认知资源 | M37 是"意识到后台存在"，这里是"主动管理前后台" |
| N12 | **情绪-认知交互觉察** | 识别情绪如何影响判断，并主动调节 | 50 关几乎完全没有情绪维度 |
| N13 | **等价变换（换一种说法解题）** | 改变问题的表达形式而不改变本质，让难题变简单 | M32 是语言翻译，这里是"把代数题画成几何图" |
