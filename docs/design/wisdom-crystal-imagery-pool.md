# 智慧结晶意象池

> 用途：M11-M50 扩写时的"意象参考库"。
> 来源：DeepSeek 原稿的 50 句儿童版智慧结晶（`/Users/danmac/AI设计/流辉美慧号游戏(1).xlsx`）。
> 处理：**只留意象，丢弃儿童口吻**——原句的比喻内核多数是精心设计的（门、画框、棋盘、流域、炼金），但"……就啦 / ……才行 / 小朋友"类表达会稀释成年读者的沉浸感。
> 使用方法：写每关的 `wisdomCrystal` 时，先从下表挑一个意象做地基，再按 [§2.4 智慧结晶写作准则](./game-design-spec.md) 重铸语言。

---

## 使用准则（三不改 + 三要改）

**三不改**（原稿的好底子）：
1. **核心比喻物**不改——画框、棋盘、流域、炼金炉、聚光灯、珍珠丝线等，这些是原稿最有价值的设计。
2. **"动作 + 对象"的结构**不改——如"把月亮和硬币一起熔炼"这种动词搭配不要拆散。
3. **与思维工具的映射**不改——意象是为工具服务的，不能为了美换成不相关的意象。

**三要改**：
1. **去掉稚化口吻**——"小朋友 / 要……才行 / ……啦 / 别难过"全删。
2. **去掉解释性尾巴**——原稿常在意象后加"这就是 XXX"，删掉，让意象自己说话。
3. **按 §2.4 两段式结构重写**——前半否定旧直觉，后半给出新直觉，用破折号或分号衔接。
