## 正交性验证

容易混淆的工具对，以及它们为什么不同：

| 容易混淆的对 | 区别 |
|------------|------|
| 01 分类 vs 19 抽象 | 分类 = "放进哪个箱子"，抽象 = "扔掉箱子标签只看形状" |
| 06 因果排序 vs 11 逻辑链重建 | 排序 = "给已知环节排对"，重建 = "发现缺了哪个环节" |
| 08 逆向推理 vs 11 逻辑链重建 | 逆向 = "从终点往回走"，重建 = "在断裂处补缺口" |
| 12 权重分析 vs 44 贡献度分析 | 权重 = "决策前：谁更重要"，贡献度 = "事后：谁造成的" |
| 14 类比迁移 vs 45 概念融合 | 迁移 = "A→B 搬解法"，融合 = "A+B→C 造新概念" |
| 17 约束识别 vs 04 多重条件满足 | 识别 = "瓶颈在哪"，满足 = "同时达成所有已知约束" |
| 18 估算 vs 26 概率 | 估算 = "大概多少"（量），概率 = "多大可能"（不确定性） |
| 20 情绪觉察 vs 37 注意力/意识 | 情绪 = "感受影响判断"，注意力 = "聚光灯照哪里" |
| 28 升维 vs 19 抽象 | 升维 = 空间/维度跃迁，抽象 = 信息密度压缩 |
| 32 符号翻译 vs 50 等价变换 | 翻译 = "跨符号系统保意义"，变换 = "换说法让题变简单" |
| 35 局部最优 vs 30 边界测试 | 局部最优 = "跳出当前山谷"，边界测试 = "故意推到极端" |
| 40 模型构建 vs 19 抽象 | 抽象 = "剥离细节"，模型 = "搭简化结构代替复杂现实" |
| 49 注意力分配 vs 37 注意力/意识 | 37 = "意识到后台存在"，49 = "主动管理前后台" |
