# Claude 版《流辉美慧号：星图守护者》— 第一阶段：器物之语（M01-M10）

> **阶段主旨**：世间万物自有秩序与表达。修复这些星星，是学会聆听事物本身的逻辑——分类是理解的起点，节奏是万物的呼吸，平衡是最古老的美学。
>
> **受众**：对思维智能成长感兴趣的人（不限年龄）
>
> **设计原则**：每个关卡是一个隐喻体验，交互本身具有诗意美感，修复即领悟。

---

## M01 沉睡的拼图星云

| 项目 | 内容 |
|------|------|
| 核心思考工具 | 分类与归属感 |
| 智慧结晶 | "每一片碎片都在寻找自己的位置。找到它属于哪里，就是理解它是什么。" |

**场景描述**：
一片星云碎成了无数带有微弱光纹的碎片，漂浮在寂静的暗蓝色虚空中。每片碎片表面隐约流动着颜色——但只有在彼此靠近时，边缘的光纹才会隐隐呼应。星云中央有一个空洞的晶体框架，像一朵尚未绽放的花骨朵，等待碎片归位后重新盛开。

**交互逻辑**：
1. 轻触碎片，观察其光纹的颜色脉动和纹路走向。
2. 将碎片缓缓推向其他碎片——若光纹频率相近，边缘会开始共鸣发光；若不匹配，光纹黯淡退却。
3. 将共鸣的碎片组合后送入晶体框架——吻合的组合会嵌入并稳定发光，错误组合会被轻柔弹回。
4. 当所有碎片归位，星云骨朵缓缓绽放成一朵旋转的光之花，花瓣是分类后的各色碎片，按色谱排列如虹。

## M02 遗忘的回声走廊

| 项目 | 内容 |
|------|------|
| 核心思考工具 | 模式识别与序列感知 |
| 智慧结晶 | "万物都有自己的节拍。安静下来，你就能听见藏在混乱中的旋律。" |

**场景描述**：
星星内部是一条悠长的晶石走廊，墙壁上嵌满了形态各异的共鸣石。走廊尽头的大门紧闭。曾经，走过这条路的人会听见石壁奏出一段引导旋律——但现在共鸣石的排列错乱，走廊只发出杂乱的嗡鸣。地面上残留着若隐若现的脚印光痕，暗示着某种行走节奏。

**交互逻辑**：
1. 点击走廊入口处仅存的一小段完好墙壁——听到一段短旋律（3-4音）。
2. 观察错位的共鸣石，每块石头点击后发出不同的音高和光色。
3. 按旋律规律重排共鸣石——不是简单记忆音序，而是发现音高的递进模式（如每隔两块升一个调）。
4. 排列正确后，莱米走过走廊，每一步踏在正确的节奏上，走廊两侧依次亮起，大门在最后一个音符落下时无声开启，露出一片星光花园。

## M03 倾斜的光之天秤

| 项目 | 内容 |
|------|------|
| 核心思考工具 | 等价判断与直觉校准 |
| 智慧结晶 | "平衡不是两边一样重，而是两边都觉得刚刚好。" |

**场景描述**：
星星核心是一架悬浮在虚空中的光之天秤，由无数光纤编织而成，此刻严重倾斜。天秤两端的托盘上放着形态迥异的"凝思晶"——有的沉重如陨石却只有拇指大小，有的轻盈如蒲公英却占据整个托盘。托盘下方的刻度不是数字，而是一道光谱：从冰蓝（过轻）渐变到暖金（恰好）再到烈红（过重）。

**交互逻辑**：
1. 拖拽凝思晶在两个托盘之间移动，观察光谱刻度的变化。
2. 关键发现：晶体的"重量"不只取决于自身——相邻晶体间会产生共振增重或抵消减重。
3. 玩家需要感知组合效应，而非简单加减。
4. 当两侧光谱都稳定在暖金区域，天秤开始极缓慢地呼吸式起伏——不是静止，而是活的平衡——光纤网络从暗淡变为通体璀璨。

## M04 淤塞的梦境河

| 项目 | 内容 |
|------|------|
| 核心思考工具 | 路径规划与流向直觉 |
| 智慧结晶 | "水不与石头争论，它只是找到绕过去的路。" |

**场景描述**：
星星表面是一条冻结的"梦境河"，河中淤积着半透明的记忆浮冰，每块浮冰内封存着一幅画面（星空、森林、海洋……）。河床上散落着可移动的导流石。河的源头有一汪温热的光泉，正在缓缓渗出，但被浮冰堵住了去路。远处河口是一个黯淡的漩涡湖，等待活水注入。

**交互逻辑**：
1. 不能直接移除浮冰（它们是珍贵的记忆），只能移动导流石改变水的走向。
2. 放置导流石引导温泉水流绕过浮冰，或温柔地将浮冰推向河岸安全区。
3. 水流有物理模拟：宽道流慢，窄道流急——急流能推开小浮冰，但会冲碎脆弱的记忆画面。
4. 河道贯通后，梦境河解冻复流，浮冰中的画面苏醒，成为河面上漂浮的微缩光景——星空在河流中流淌。

## M05 迷途的导航仪

| 项目 | 内容 |
|------|------|
| 核心思考工具 | 参照系建立与校准 |
| 智慧结晶 | "在判断别人有没有迷路之前，先确认你自己脚下站的是不是稳固的地面。" |

**场景描述**：
古旧的天文台内部，中央是一具多层嵌套的青铜星盘，每层圆环刻着不同的天体符号，但所有层都错位了。星盘周围漂浮着几个真实的微缩天体模型——一颗小太阳、一弯月牙、一簇星团——它们是"绝对参照物"。天文台穹顶有一个空洞，等待星盘校准后投射出航路。

**交互逻辑**：
1. 将微缩天体模型靠近星盘——与之对应的符号层会微微发光，提示对齐方向。
2. 旋转每一层圆环，让符号与真实天体方位对齐。
3. 关键：必须从最内层（太阳）开始校准，因为外层的正确位置取决于内层——如果参照系本身偏了，所有校准都是错的。
4. 全部对齐后，星盘中心射出一道全息星航道图穿过穹顶空洞，照亮了通往下一片星域的路。

## M06 断弦的共鸣钟

| 项目 | 内容 |
|------|------|
| 核心思考工具 | 排除法与逆向筛选 |
| 智慧结晶 | "找到答案的方法不只有'这是什么'，还有'这不是什么'。去掉所有不对的，剩下的就是真相。" |

**场景描述**：
两颗造型如古钟的水晶星星，面对面悬挂在光链上，之间漂浮着数面半透明的声障——有的是实心壁，有的有特定形状的镂空。两颗钟沉默已久。它们的钟面有截然不同的纹路：一颗是水波纹，一颗是锋棱纹。

**交互逻辑**：
1. 轻敲水波钟——发出圆润的环形波，观察它遇到每面声障的反应：有的镂空能让环形波通过，有的将其完全挡回。
2. 轻敲锋棱钟——发出尖锐的锯齿波，通过规律不同。
3. 玩家需要旋转和重排声障，为两种波各清出一条无障碍通道——方法不是"找到对的路"，而是"排除所有挡路的障碍"。
4. 双通道贯通的瞬间，两钟同时鸣响，声波在中点交汇，化为一朵声音之花——可见的、旋转的、美丽的驻波图案。

## M07 错位的星座织锦

| 项目 | 内容 |
|------|------|
| 核心思考工具 | 模式匹配与节奏同步 |
| 智慧结晶 | "倾听每颗星星心跳的节奏，就能把它放回属于它的旋律里。" |

**场景描述**：
一幅星座织锦（兔子轮廓）上的数颗星星脱落了，如失忆的萤火虫在周围漫游。织锦上留有空位，每个空位按独特的节奏闪烁着微光——像心跳。游离的星星也各有自己的脉动节奏，但它们不知道自己属于哪里。

**交互逻辑**：
1. 点击织锦上的一个空位——感受它的闪烁节奏（视觉+轻柔音效）。
2. 驾驶飞船靠近一颗游离星，观察它的脉动。
3. 当星星的脉动与空位的节奏同步时，轻推它——星星会被一条光丝轻柔地牵引回位，嵌入织锦并稳定发光。
4. 若节奏不匹配，星星会微微颤抖后飘回原处——不是惩罚，而是"还不是这里"的温柔提示。
5. 所有星星归位后，织锦上的兔子轮廓完整亮起，星星们开始以各自的节奏协同闪烁，构成一首无声的视觉交响。

## M08 闭锁的虹膜之门

| 项目 | 内容 |
|------|------|
| 核心思考工具 | 多条件复合满足 |
| 智慧结晶 | "一把好的锁不只需要正确的钥匙，还需要在正确的时刻、用正确的方式转动。" |

**场景描述**：
星星核心是一道巨大的虹膜门——如同瞳孔般由多层金属花瓣交错锁闭，每层花瓣上有不同形状的缺口（三角、方形、圆形）。门后透出隐约的温暖光芒。周围漂浮着与缺口对应的光之钥碎片。

**交互逻辑**：
1. 旋转每层花瓣，尝试让所有缺口在某个角度对齐成一条贯穿的复合通道。
2. 仅对齐缺口不够——还需拾取光之钥碎片，按形状嵌入通道的对应层。
3. 嵌入顺序也有讲究：外层三角 → 中层方形 → 内层圆形，因为内层碎片只有在外层已嵌入后才能通过通道抵达。
4. 所有钥匙就位，虹膜如苏醒的眼睛般柔和张开，露出内部缓缓旋转的星核——一颗纯净的光之心脏，脉动着将温暖向整颗星星输送。

## M09 淤塞的星光过滤塔

| 项目 | 内容 |
|------|------|
| 核心思考工具 | 流程排序与因果链 |
| 智慧结晶 | "先吸走铁屑，再筛掉沙砾，最后溶解粘液——顺序对了，浊流自清。" |

**场景描述**：
星星的能量中枢——一座多层旋转过滤塔。混浊的"思维残渣"涌入：金属碎屑、粗砺岩粒、粘稠的彩色胶体。过滤塔有三层可旋转的净化机制：磁力层、滤网层、溶剂层。当前顺序混乱，残渣卡在错误的层中，整座塔噗噗作响。

**交互逻辑**：
1. 观察混合残渣的成分——金属会被磁力吸附，岩粒会被滤网拦截，胶体会被溶剂分解。
2. 旋转塔层调整顺序——错误的顺序会导致连锁堵塞（如先用溶剂，胶体溶解后粘住金属碎屑，反而更堵）。
3. 正确顺序：磁力（先除金属）→ 滤网（再拦岩粒）→ 溶剂（最后化胶体）。每层清除成功时会发出满足的机械吟唱。
4. 全部通畅后，过滤塔高速运转，混浊输入化为三道分色纯净光流从出口喷涌——一道金色、一道银色、一道虹色——交织成星星新的能量经络。

## M10 万花筒的无名之歌

| 项目 | 内容 |
|------|------|
| 核心思考工具 | 开放探索与主观创造 |
| 智慧结晶 | "没有唯一正确的图案。你愿意停留凝视的那一刻，就是这颗星星等待的答案。" |

**场景描述**：
进入这颗星星，整个视野化为一架巨大的万花筒。镜片角度错乱，彩色光屑如迷失的蝴蝶在镜面间无序飞舞。屏幕边缘有几根古铜控制杆（调节镜片角度）和一个呼吸孔（吹气扰动光屑、吸气凝聚光屑）。没有分数，没有倒计时，没有"正确答案"的提示。

**交互逻辑**：
1. 自由推拉控制杆——镜片角度改变，折射出完全不同的对称图案。
2. 轻触呼吸孔——吹气让光屑飞散重组，吸气让光屑聚拢凝结。
3. 没有预设的胜利图案。当玩家创造出自己觉得和谐、美丽的图案，并停下操作静静欣赏一段时间——星星会感应到这份"驻足的专注"。
4. 万花筒图案开始缓慢自转，光屑凝固为晶体，这个由玩家创造的图案成为星星永恒的面貌。一行小字浮现："这颗星星，以你的凝视命名。"

## 第一阶段设计理念

**认知递增线索**：从被动观察（M01 分类）→ 主动感知（M02 模式）→ 直觉判断（M03 平衡）→ 间接策略（M04 路径）→ 建立参照（M05 校准）→ 逆向思维（M06 排除）→ 节奏共鸣（M07 同步）→ 复合条件（M08 多重匹配）→ 因果排序（M09 流程）→ 自由创造（M10 开放式）。

**难度递增体现**：操作维度从单一到复合，思维模式从直觉到分析再到超越分析（M10 没有标准答案）。

**诗意贯穿**：每一关的修复动画都是一个"美的瞬间"——星云绽放为花、走廊奏响旋律、天秤学会呼吸、河流承载记忆、星盘投射航路、声波凝结为花、织锦复现星座、瞳孔苏醒为心脏、浊流化为三色光、万花筒以凝视命名。
