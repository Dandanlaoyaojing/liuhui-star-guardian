## M01 记忆齿轮的卡顿

**思维工具**：分类与归纳
**工具一句话**：按属性把事物分成组

### 场景

一颗齿轮状的星星，内部如同巨型怀表的心脏。数百枚"记忆碎片"——半透明的光之薄片——被杂乱地塞进齿槽，让精密的齿轮完全卡死。每枚碎片有两个属性：颜色（红/蓝/黄）和形状（圆/三角/方）。

齿轮侧面有三道彩色光缝——过滤器插槽。远处漂浮着三面棱镜状的"色彩过滤器"，分别映着红、蓝、黄。齿轮中心空出一个圆形收纳盘，被分成九格（3色×3形），每格底部刻着对应的图标提示。

整颗星星暗沉无光，只有碎片在齿槽中微弱地闪烁。

### 交互

1. **装载过滤器**：将红色过滤器拖入插槽。一道柔和的红色光幕笼罩齿轮——所有非红色碎片瞬间变暗变透明，只剩红色碎片在齿槽中清晰发亮。世界被简化了。

2. **吸取归类**：用流辉美慧号的吸盘（长按拖拽）逐一吸出发亮的红色碎片。将圆形碎片拖入收纳盘的"红+圆"格——吸附时发出清脆的"叮"声，碎片自动旋转归位。三角和方形同理。放错格会被轻柔弹回，碎片微微摇头。

3. **轮换视角**：拔出红色过滤器，插入蓝色。世界再次被简化——只剩蓝色碎片发亮。重复吸取归类。最后换黄色。

4. **关键理解**：玩家自然体会到——面对混乱时，先选一个维度（颜色），把其他信息暂时屏蔽掉，世界就变得可处理了。这就是分类。

### 修复动画

最后一枚碎片归位，收纳盘"啪"一声闭合。齿轮咬合，开始缓慢旋转。收纳盘中九种碎片被离心力甩出——但不再是碎片，而是九道色彩分明的光带，螺旋上升，编织成齿轮星星新的光冕。星光从暗沉变为通体辉煌。

**智慧结晶**："把混乱的朋友们，送回各自颜色的家，世界就有序啦。"

## M02 沉默的共鸣钟

**思维工具**：模式匹配与排除
**工具一句话**：观察规律，排除不匹配的，找到能通过的

### 场景

两颗钟形水晶星星在深空中面对面悬挂，由一根纤细的光链相连。左钟表面是流动的水波纹理，右钟是锐利的锋棱纹理。两钟之间的空间里，漂浮着六面半透明的"声障"——有的是实心光壁，有的上面镂着三角形孔洞，有的有星形网格。

空间是安静的。钟无法发出声音，因为所有路径都被声障封锁了。

### 交互

1. **实验观察**：轻敲左钟（水波纹），它发出一圈柔和的环形光波向右传播。光波遇到实心壁——被完全挡回；遇到镂空三角——穿过了；遇到星形网格——被打散。轻敲右钟（锋棱纹），发出锯齿形光波。锯齿波能穿过星形网格，但被三角孔挡住。

2. **发现模式**：环形波只能穿三角孔，锯齿波只能穿星形网格。实心壁谁也过不去。这就是"模式"——波的形状必须和孔的形状匹配。

3. **排除障碍**：拖拽和旋转声障，为两种波各清出一条通道。关键不是找路，而是**排除挡路的东西**——把所有不让环形波通过的障碍移开，把所有不让锯齿波通过的障碍移开。

4. **工具应用**：先分析"什么能通过什么"（模式匹配），再把"不能通过的"全部排除，剩下的自然就是路。

### 修复动画

双通道贯通瞬间，两钟同时鸣响。环形波和锯齿波在中点相遇，不是相消，而是交织成一朵旋转的驻波图案——像一朵由声音织成的、不断变幻的光之花，悬浮在两钟之间。光链开始脉动，将两颗钟的心跳同步。

**智慧结晶**："为不同的声音，找到它能穿过的门，它们就能合唱了。"

## M03 迷失的导航罗盘

**思维工具**：系统校准与参照
**工具一句话**：先找到可靠的基准点，再用它去校准其他所有东西

### 场景

星星内部是一座幽暗的古观测台。中央悬浮着一具五层嵌套的青铜星象罗盘——每层是一个可独立旋转的圆环，刻着天体符号（日、月、星、云、风），但全部错位。罗盘指针在五层矛盾的牵引下剧烈抖动，无法定向。

观测台穹顶有一个圆形天窗，透进真实的星光。场景中漂浮着五个微缩天体模型：一颗小太阳、一弯月牙、一颗明星、一朵云、一个风向标。它们是"真实的参照物"——不随罗盘错位而改变。

### 交互

1. **建立参照**：将小太阳模型移近罗盘——刻有太阳符号的最内层圆环微微发光，提示对齐方向。旋转该层使太阳符号指向小太阳模型。"咔"，第一层锁定。

2. **逐层校准**：用月亮模型校准第二层，明星校准第三层，云校准第四层。每层校准时，指针的抖动会减弱一分。

3. **关键理解**：**必须从最内层开始**。如果先校准外层，当内层后来调整时，外层的"正确"位置也跟着变了——参照系是有层级的，底层偏了，上面全偏。这是这关最核心的工具：**先校准你的尺子，再去量世界。**

4. **最后一步**：四层校准后，用飞船引擎对风向标轻吹一口气，观察它稳定后的指向，手动旋转最外层空白环对齐。

### 修复动画

五层全部锁定，指针瞬间静止，指向穹顶天窗。罗盘核心射出一束凝聚的全息星航道图——一条由光线编成的三维路径穿过天窗，照亮了星星之外的一片新星域。观测台内壁上沉睡的星图全部亮起。

**智慧结晶**："先对准真正的太阳和月亮，罗盘才会告诉你正确的方向。"

## M04 熵增的星沙漏

**思维工具**：路径规划与分拣
**工具一句话**：在约束条件下，为不同的东西设计不同的路径

### 场景

一颗沙漏形星星。上半部淤积着海量发光的"星沙"——细如光尘，混乱地闪烁。三种闪烁频率清晰可辨：急促快闪（金色）、稳定中闪（银色）、缓慢慢闪（青色）。

沙漏中央通道被几块形态各异的隔板部分阻塞——有的可以滑动，有的可以移除，有的是固定的。底部有三个收集槽，入口各有不同密度的滤网：细密滤网（对应快闪）、中密滤网（对应中闪）、稀疏滤网（对应慢闪）。

星沙一粒都流不下去。沙漏停摆了。

### 交互

1. **观察对应关系**：点击底部收集槽，滤网图案闪烁——玩家需要把快闪沙送到细密槽、中闪沙送到中密槽、慢闪沙送到稀疏槽。

2. **设计路径**：拖动/移除隔板改变通道结构。关键洞察——通道的宽窄会影响流速：
   - 窄道：沙流快，只有轻盈的快闪沙能快速通过
   - 宽道：沙流慢，适合慢闪沙从容流淌
   - 弯道：会减速，可以用来分流

3. **试错调整**：放开一小段通道，观察星沙实际流动。金色快闪沙是不是进了正确的槽？没有就调整隔板。这是**路径规划**——先观察规律，再设计路线，在约束下把不同的东西分别送到该去的地方。

4. **不能混流**：如果路径设计不当，三种沙混着流入同一个槽，收集槽会闪红——需要重新调整。

### 修复动画

三种星沙各归其位，沙漏发出低沉的满足声。然后整具沙漏无声翻转——星沙从底部槽中以三道色彩分明的光流重新涌回上半部，金银青三色交织成螺旋上升的光柱。沙漏开始永恒的优雅翻转，星沙在内部流淌如极光。

**智慧结晶**："为跑得快和走得慢的沙粒，设计不同的滑梯，它们就能愉快地流动了。"

## M05 冻结的星之河

**思维工具**：间接策略与资源管理
**工具一句话**：不直接解决问题，而是利用环境中的力量间接达成目标

### 场景

星星表面，一条"光之液"河道完全冻结，河面凝固成蓝白色的冰层。河上矗立着五座巨大的"情绪浮冰"——不透明、沉重、嵌入冰面。每块浮冰内部封存着不同颜色的气体（红/蓝/中性白）。

流辉美慧号的尾部伸出一面可调角度的聚光透镜——能将星光聚焦为热束。河床两侧隐藏着四组导流板控制杆。河的源头有一汪温泉般的光之泉眼，缓缓渗出热流，但被冰层封住了出路。远处河口是一片黯淡的环形湖。

### 交互

1. **直接尝试**：用透镜照射浮冰。冰融化了——但释放出气体。**红气让周围河水沸腾激荡**（危险，可能冲毁河岸设施）。**蓝气让局部河水重新冻结**（适得其反）。白气无害消散。

2. **发现间接策略**：不能硬融所有冰。先找那块封存白色中性气体的浮冰融化——安全地打开一段河道。

3. **借力打力**：操作导流板控制杆，引导已解冻的温热河水流向另一块浮冰的底部——用水的温度从下方慢慢侵蚀冰基，而不是用透镜从上方暴力融化。这样气体被水压稀释，不会造成剧烈反应。

4. **资源管理**：透镜的聚光能量有限（通过亮度条显示），不能无限使用。必须精打细算——把直接融化留给安全的白气冰，把危险的红气冰和蓝气冰交给间接的水流策略。

5. **核心领悟**：最聪明的做法往往不是直接对目标下手，而是改变环境条件，让问题自己解开。

### 修复动画

最后一块浮冰被河水无声地托起、推离河道，搁浅在岸边（没有被摧毁——它们只是不该待在河里）。光之液从泉眼涌出，沿着疏通的河道奔流而下，注入环形湖。湖面亮起，倒映出整片星空。河道两岸的冻土开始长出微弱的光之草。

**智慧结晶**："有时，用流动的水去推开冰，比用光去融化它，更聪明。"

## M06 错位的星座拼图

**思维工具**：模式识别与节奏同步
**工具一句话**：观察事物的节奏规律，把节奏匹配的东西配对

### 场景

一幅巨大的星座织锦（莱米家族的兔子星座）漂浮在深空中——一个由光线勾勒的兔子轮廓，但轮廓上有七个空位，应有的星星脱落了。它们像迷路的萤火虫在周围漫游，每颗都在独自闪烁。

织锦上的七个空位也在闪烁——但每个空位有自己独特的节奏：有的是"闪-闪-停-闪-闪-停"的均匀二拍，有的是"闪-停-停-闪-停-停"的舒缓三拍，有的是不规则的心跳式脉动。

游离的星星也各有节奏。但它们漂浮不定，节奏在移动中会微微漂移——需要靠近了才能听准。

### 交互

1. **感受节奏**：点击织锦上的一个空位——它放大显示，闪烁节奏变得清晰，同时发出对应的轻柔音律（视觉+听觉双通道）。

2. **寻找匹配**：驾驶飞船靠近一颗游离星。当飞船足够近时，该星的脉动节奏变得清晰可辨。玩家需要判断：这颗星的节奏和刚才那个空位的节奏是同一个模式吗？

3. **尝试配对**：如果认为匹配，轻推星星向空位移动。节奏真正同步时，一条光丝从空位伸出，温柔地将星星牵引归位——嵌入的瞬间，星星和空位的节奏完美合一，亮度翻倍。如果不匹配，星星会微微颤抖然后飘回——不是惩罚，是温柔的"不是这里"。

4. **渐进难度**：前几颗星节奏差异明显（快vs慢），后几颗越来越接近，需要更细致的辨别。最后一颗的节奏是前面两颗的叠加——模式识别的复合考验。

### 修复动画

最后一颗星归位，七颗星同时亮起。它们不是齐闪——而是以各自的节奏协同脉动，如同一支无声的小型乐队在合奏。织锦上的兔子轮廓被这七颗星的光线填满，投射出一个缓缓旋转的立体兔子光影——它眨了眨眼。

**智慧结晶**："倾听每颗星星心跳的节奏，就能把它放回属于它的旋律里。"

## M07 闭锁的虹膜星门

**思维工具**：多重条件满足与精准匹配
**工具一句话**：多个条件必须同时满足，每个细节都不能差

### 场景

星星核心是一道巨大的虹膜门——由四层可独立旋转的金属花瓣交错咬合而成，像一只紧闭的机械瞳孔。每层花瓣上有不同形状的缺口：最外层是三角形缺口，第二层是方形，第三层是圆形，最内层是星形。

门缝中透出温暖的橙色光——核心就在后面，但门死死锁闭。

周围的空间中漂浮着四种形状的"光之钥"碎片——三角、方形、圆形、星形，各一枚。每枚碎片发出与自身形状对应的微弱脉冲。

### 交互

1. **对齐缺口（条件一）**：用飞船的机械指旋转每层花瓣。目标是让四层缺口在同一角度贯穿对齐——形成一条从外到内的复合通道。难度在于旋转一层时，相邻层会因齿轮咬合而微微联动，需要反复微调。

2. **嵌入钥匙（条件二）**：缺口对齐后，通道呈现出从外到内"三角→方→圆→星"的叠套形状。拾取三角光钥嵌入最外层缺口——吻合，锁定。然后方形——吻合。圆形——吻合。

3. **顺序约束（条件三）**：内层钥匙必须穿过外层缺口才能到达。如果先试图嵌入星形（最内层），它会被外面三层未就位的钥匙挡住。**必须从外向内依次嵌入。** 这就是"多重条件"——不仅每把钥匙要形状对，对齐要精准，顺序也不能错。

4. **容错提示**：放错钥匙，该层花瓣会微微振动并弹回钥匙——不是惩罚，而是反馈"条件不满足"。

### 修复动画

四把钥匙全部就位，虹膜门发出一声深沉的机械叹息。四层花瓣像苏醒的瞳孔般缓缓向外绽开，露出内部——不是空洞，而是一颗跳动的光之心脏，以沉稳的节律脉动，每一次搏动都向整颗星星的脉络输送暖光。星星从内到外渐次亮起，如同血液重新流过沉睡的躯体。

**智慧结晶**："只有当每一层的锁孔都对准，并且放入形状正确的钥匙，大门才会欣然打开。"

## M08 失调的谐波天平

**思维工具**：动态平衡与系统调节
**工具一句话**：平衡不是静止，而是找到让系统持续和谐振荡的那个点

### 场景

两颗星星由一条粗壮的光带相连，构成一架宇宙天平——但严重倾斜，一高一低，光带绷得快要断裂。两颗星表面各有几枚可拖动的"概念砝码"——造型各异（羽毛、锚、书册、水滴），代表不同的重量和特性。

光带正中有一个可滑动的支点标记。天平底座有一个弧形仪表：左端标"僵死"（完全静止），右端标"失控"（剧烈摆动），中间有一小段绿色区域标"和谐"。

指针此刻指在"僵死"附近——一端太重压住了另一端。

### 交互

1. **初步调节**：拖动砝码在两颗星之间移动。把锚（重）从低端移到高端——天平开始缓慢回正。但如果移太多，天平翻向另一边，从"一边倒"变成"另一边倒"。

2. **发现动态目标**：仪表的"和谐"区不是指天平水平静止——而是指天平在做**微小的、规律的起伏运动**，像呼吸。完全水平反而落在"僵死"区。

3. **精细调节**：滑动支点位置改变力臂比例。结合砝码分布和支点位置，让天平进入一种微微摆动的稳态——不是静止的平衡，而是**活的平衡**。

4. **核心领悟**：真正的平衡不是让两边一样重然后冻住。而是让系统在一个范围内自由运动，但不失控。这在情绪管理、人际关系、甚至生态系统中都是同一个工具。

### 修复动画

指针进入"和谐"绿区的瞬间，两颗星开始以光带中点为轴，缓慢、优雅地交替升降——像宇宙在呼吸。每次升到最高点的那颗星会短暂地额外发亮，然后把光传递给下沉中的另一颗。光带不再绷紧，而是柔韧地随运动波动，发出和弦般的低频振鸣。

**智慧结晶**："平衡不是一动不动，而是像跷跷板一样，有来有回，唱着快乐的歌。"

## M09 淤塞的星光过滤器

**思维工具**：流程优化与顺序逻辑
**工具一句话**：做事的顺序决定效率，先处理什么后处理什么至关重要

### 场景

星星的能量中枢——一座三层的旋转过滤塔，矗立在汇聚的管道交叉口。混浊的"思维残渣"从四面涌入：闪着金属光泽的铁屑、灰褐色的粗砺岩粒、以及粘稠的彩色胶体（像熔化的糖）。三种残渣混合在一起，堵塞了过滤塔。

过滤塔的三层各有不同的净化机制，可以通过旋转塔身改变上下顺序：
- **磁力层**（蓝色环）：能吸附金属
- **滤网层**（银色环）：能拦截大颗粒
- **溶剂层**（绿色环）：能溶解胶体

当前顺序是乱的——溶剂在最上面，磁力在最下面。残渣涌入后一团糟。

### 交互

1. **观察失败**：当前顺序下启动过滤器——胶体先被溶剂层溶解，变成黏液流下来，把下面滤网层的网眼全糊住了，岩粒和铁屑彻底过不去。过滤塔发出痛苦的嗡鸣，红灯亮起。

2. **分析问题**：三种残渣的特性不同——铁屑只有磁力能吸，岩粒只有滤网能拦，胶体只有溶剂能化。但**顺序错了后果会连锁**：如果胶体先被溶解，黏液会污染后续层；如果岩粒先被拦住但铁屑没被吸走，铁屑会刮坏滤网。

3. **推理正确顺序**：
   - 第一层应该是磁力层——先吸走铁屑，因为铁屑会损坏其他层
   - 第二层应该是滤网层——再拦住岩粒，因为岩粒会堵住溶剂层
   - 第三层应该是溶剂层——最后溶解胶体，因为黏液不会影响已经完成工作的上层

4. **旋转调整**：拖动塔层旋转位置。每次调整后可以试运行一小批残渣，观察效果。

### 修复动画

正确顺序设定完成，过滤塔全速运转。每层依次亮起自己的颜色——蓝色磁力层嗡嗡作响吸走铁屑，银色滤网层轻轻振动筛去岩粒，绿色溶剂层无声溶化胶体。三层之后，从塔底出口喷涌出的不是残渣，而是三道纯净光流——金色、银色、虹色——交织成星星新的能量经络，为整颗星星注入活力。

**智慧结晶**："先吸走铁块，再筛掉大石头，最后溶解黏黏糖——按对的顺序来，过滤器就不会累。"

## M10 失序的星空万花筒

**思维工具**：探索、创造与开放式解决
**工具一句话**：没有标准答案，你自己定义什么是"好"

### 场景

进入这颗星星的瞬间，莱米的整个视野被接管——化为一架巨型万花筒的内部。三面巨大的镜片组成三角形通道，彩色光屑（红、蓝、绿、金、紫……）如失重的蝴蝶在镜面之间无序飞舞。镜片角度各自歪斜，折射出的图案是混乱的、破碎的、不成形的。

屏幕下方有三根古铜色控制杆：
- **左杆**：调节左镜片角度（0°-180°）
- **中杆**：调节右镜片角度
- **右杆**：调节底镜片角度

屏幕右侧有一个古铜呼吸孔——按住吹气让光屑飞散重组，按住吸气让光屑聚拢凝结。

没有分数条。没有目标提示。没有倒计时。画面中央只有一行淡淡的字："找到你的图案。"

### 交互

1. **自由实验**：推拉控制杆——镜片角度每变一点点，万花筒图案就彻底改变。从混沌碎片变成对称的六角雪花，再变成旋转的涡流，再变成嵌套的同心圆。没有两个角度组合产出一样的图案。

2. **扰动与凝聚**：轻触呼吸孔——吹气让光屑炸开重新分配颜色比例，吸气让光屑聚拢形成更密集的核心。这改变的不是结构而是"颜料"——同样的镜片角度，不同的光屑分布会产生截然不同的色彩感受。

3. **没有"正确答案"**：这关没有预设的胜利图案。系统在等待的是玩家的**主观判断**——当你创造出一个自己觉得和谐的、美丽的、愿意停下来凝视的图案时，停止操作。

4. **驻足即完成**：当玩家停下所有操作并持续凝视画面一段时间（约 5-8 秒不触碰任何控制），星星会感应到这份"专注的驻足"。

### 修复动画

万花筒图案开始极其缓慢地自转——不是破坏，而是将这一刻的美凝固为运动中的永恒。光屑逐渐结晶，从飘浮的粒子变成固体的微型宝石，嵌入镜片表面。万花筒不再是一个需要操控的装置，而变成了一颗自行旋转的、永远展示着玩家创造的图案的宝石星星。

画面底部浮现一行字："这颗星星，以你的凝视命名。"

然后是智慧结晶：

**智慧结晶**："没有唯一'正确'的图案。你感到美丽和开心的那个，就是最好的答案。"
